# sany  我的第一个代码学习项目
## 完整流程总结
| 步骤 | 命令 | 作用域 | 说明 |
|------|------|--------|------|
| 1 | `git init` | 本地 | 初始化仓库 |
| 2 | `git add` | 本地→暂存区 | 暂存文件 |
| 3 | `git commit` | 暂存区→本地仓库 | 创建提交 |
| 4 | `git branch -M` | 本地 | 重命名分支 |
| 5 | `git remote add` | 本地↔远程 | 添加远程地址 |
| 6 | `git push` | 本地→远程 | 推送代码 |
